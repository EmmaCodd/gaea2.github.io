css居中各种实现

  • css

水平居中

1、行内元素（img、span等）可以用text-align:center针对它的父元素设置水平居中。

2、块级元素（div、p、form、ul等）可以将要居中的元素的margin-left和margin-right都设为auto，也就是margin:0 auto，也可以写作margin:0 auto 0 auto

以上两种方法都是用来水平居中的，但对浮动元素或绝对定位元素无效。

3、块级元素可以利用css3的justify-content属性实现水平居中。设置其父元素的display属性为flex ，justify-content属性为center即可。

垂直居中

1、单行文字垂直居中，让line-height的值与height相等即可。

2、对已知高度的块级元素用绝对定位来实现垂直居中。

原理是将该元素的top的值设为50%，这时元素比垂直居中往下偏了自身高度的一半，所以需要一个负的margin-top的值来向上挪动自身高度一半的距离，从而达到垂直居中的位置。

3、对于多行行内元素，可以在父级元素中通过display:table-cell;vertical-align:middle; 将它模拟成一个表格单元格，就可以很方便的实现垂直居中。
其中最关键的是最外层div设置样式：display: table;

内层div设置样式： display: table-cell;vertical-align: middle;
