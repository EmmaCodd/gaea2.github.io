Markdown 语法学习

  • markdown

概念

Markdown is a way to style text on the web. You control the display of the document; formatting words as bold or italic, adding images, and creating lists are just a few of the things we can do with Markdown. Mostly, Markdown is just regular text with a few non-alphabetic characters thrown in, like # or *.

Markdown简洁优雅，适合文艺工作者在线写作使用，当然github等社区也用Markdown作为主流编辑方式。Markdown还兼容html，多数html标签都可以直接在文档中撰写。而像div、table、p等区块标签，需要在前后加上空行与其它内容区隔开。

语法说明

标题Headers

在行首插入1 到 6 个 # ，对应到标题 1 到 6 阶。

# This is an <h1> tag
## This is an <h2> tag
###### This is an <h3> tag
强调Emphasis

用星号 * 或下划线 _ 作为强调的符号，被 * 或 _ 包围的语句会变为斜体，被两个 * 或 _ 包围的语句会被转成粗体。当然也可以两者结合。

*This text will be italic*
_This will also be italic_

**This text will be bold**
__This will also be bold__

_You **can** combine them_
列表Lists

无序列表使用星号、加号或是减号作为列表标记，可以嵌套。

* Item 1
* Item 2
  * Item 2a
  * Item 2b

有序列表则使用数字接着一个英文句点，也可以嵌套。

1. Item 1
2. Item 2
3. Item 3
   * Item 3a
   * Item 3b

需要注意的是，有序列表标记上使用的数字不会影响输出的 HTML 结果。

1.  gaea2
1.  mikoto
1.  cloud

对应html source是：

<ol>
<li>gaea2</li>
<li>mikoto</li>
<li>cloud</li>
</ol>

对应html preview是：

1.gaea2
2.mikoto
3.cloud
图片Images

一个感叹号 !，然后一个方括号，里面放图片的描述文字，然后一个小括号，放上图片的地址。

![raligun](/images/raligun.jpg)
Format: ![Alt Text](url)
链接Links

一个方括号，里面放链接描述文字，然后一个小括号，里面放链接地址。与插入图片的语法仅缺少一个感叹号。

[example](http://example.com)
区块Blockquotes

在每行的最前面加上 > ，可以嵌套。

As Kanye West said:

> We're living the future so
> the present is our past.
代码code

1、对于行内代码，可以将代码用反引号`包围起来。（反引号在键盘Tab上方、1键的左方

I think you should use an
`<addr>` element here instead.

2、对于多行代码，可以通过缩进四个空格来实现。（本文中多采用这种方式

代码区块：
    <p>hello <b>world</b>
    </p>

3、对于多行代码，也可以在代码段前加三个反引号，然后写出代码类型，在代码区快最后再接三个反引号。这种做法还能使代码高亮。

可以在一行中用三个以上的星号建立一个分隔线，星号间可以有空格。
