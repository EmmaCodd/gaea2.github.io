win下Hexo博客的搭建过程

  • git
  • hexo

Hexo概念

Hexo 是一个轻量的基于Node.js的静态博客框架，可以生成静态网页托管在github 上,作者是台湾大学生tommy351。

快速、简单且功能强大的 Node.js 博客框架。A fast, simple & powerful blog framework, powered by Node.js.

GitHub Pages概念

GitHub Pages 可以认为是用户编写的、托管在github 上的静态网页，用于介绍托管在github上的Project或者搭建网站。github提供模板，允许站内生成网页，也允许用户自己编写网页，然后上传。生成的网站的默认域名是 username.github.io 或者 username.github.io/project-name ，也可以自定义域名。

在win下安装Hexo

1、安装git的windows版本

下载地址https://git-for-windows.github.io/，按默认选项安装即可。

安装完成后，在开始菜单里找到“Git”->“Git Bash”，出现如下图所示，说明安装成功。

2、安装Node.js

到官网下载即可 https://nodejs.org/en/download/

安装完成后在命令行输入以下命令验证是否安装成功：

node -v
npm -v

3、安装Hexo

在任意位置点击鼠标右键，选择Git Bash

输入npm install -g hexo

安装成功后，进入自己创建的某个文件夹（如”E:\blog”），右键选择Git Bash 执行hexo init

然后安装依赖npm install ，继续在Git Bash命令行输入

hexo g
hexo s

访问localhost:4000 即可看到效果。

4、将本地Hexo部署到github

首先，你得有个github账号，在https://github.com/可以注册。

在github上建立一个仓库，命名为username.github.io ，其中的username为你的github用户名。如我的github用户名是Gaea2，对应仓库名应该是 gaea2.github.io

到刚创建的仓库页面，复制对应https，例如https://github.com/Gaea2/gaea2.github.io.git

打开E:\blog\_config.yml文件，修改deploy的repository为刚才复制的地址。保存。

deploy:
  type: git
  repository: https://github.com/Gaea2/gaea2.github.io.git
  branch: master

接下来执行npm install hexo-deployer-git –save 以安装 hexo-deployer-git，否则可能出现”Deployer not found：git”的错误。

最后在Git Bash中输入

hexo g
hexo d
5、完成部署

在浏览器访问https://username.github.io/即可看到你的博客。
