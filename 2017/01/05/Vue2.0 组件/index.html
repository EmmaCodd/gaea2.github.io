  • javascript

组件是可复用的，带有特定功能的视图。若项目中需反复用到一些内容，可以用组件来完成内容的封装和复用。

注意
  • 组件不能直接返回多个要渲染的元素，将要渲染的元素放在同一个容器中，如div

  • 允许组件嵌套

  • 全局组件和局部组件的区别

    • 通过Vue.component 创建的组件是全局组件，它可以直接在任何一个组件或者模板内容中去使用
    • 通过new Vue({components:{}}) 创建组件的组件是局部组件，它只能直接用在父模板中
  • 提到组件，想到封装和重用，其次想到组件树（帮助理清楚一个应用程序的组件的基本架构）
