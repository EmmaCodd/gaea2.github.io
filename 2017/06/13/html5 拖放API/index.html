html5 拖放API

  • html5

html5 中，提供了直接支持拖放操作的API。新的拖放API 支持浏览器与其他应用程序之间的互相拖动。

Drag：拖动，Drop：释放。拖动：源对象 -> 目标对象。

拖放源对象时可触发事件
dragstart 拖动开始
drag 拖动中
dragend 拖动结束
整个拖动过程: dragstart一次+ drag多次+ dragend一次

拖放目标对象时可触发事件
dragenter 拖动着进入
dragover 拖动着悬停在上方
dragleave 拖动着离开
drop 拖动释放
整个过程1: dragenter一次+ dragover多次+ dragleave一次
整个过程2: dragenter一次+ dragover多次+ drop一次
注意：
必须阻止dragover 的默认行为(只要悬停立即离开) -> e.preventDefault();，drop 才可能触发。
