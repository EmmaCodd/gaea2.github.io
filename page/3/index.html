2016-09-24

gulp前端自动化

与grunt相比，gulp使用node.js串流(streams)使建构更快速，效率更高语法更自然,不需要编写复杂的配置文件。gulp利用来源档案当作输入，串流到一群外挂(plugins)，最后取得输出的结果，并非像grunt一样配置每一个外挂的输入与输出。

Installing Gulp

和grunt一样，它们都基于node.js，需要先搭建node环境。然后在命令行中cd到你的项目文件夹，在管理员权限下执行npm install -g gulp以全局安装gulp包。安装完成后，输入gulp -v查看gulp版本号验证gulp是否被正确安装。
